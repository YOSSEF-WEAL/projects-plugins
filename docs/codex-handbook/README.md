# Codex Handbook (Malka Projects Plugin)

هذا الفولدر مرجع ثابت بيننا لأي شغل لاحق على الإضافة.

## الهدف
- شرح المشروع بسرعة قبل أي تعديل.
- توثيق هيكل الملفات ووظيفة كل فولدر.
- توثيق شكل البيانات (CPT / Taxonomy / Meta / Options).
- توثيق تدفق الواجهة (Archive -> Filter -> Single).

## الملفات
- `01-project-overview.md`: ملخص سريع للمشروع والحالة الحالية.
- `02-folder-map.md`: كل فولدر بيعمل إيه.
- `03-data-model.md`: الداتا بيز والحقول والمعاني.
- `04-runtime-flow.md`: تدفق العمل في الفرونت والإدمن.
- `05-change-log.md`: سجل مختصر للتعديلات المهمة.

## قواعد تحديث هذا الفولدر
- أي ميزة جديدة لازم يتحدث لها ملف واحد على الأقل هنا.
- لو تغير اسم meta key أو option key لازم يتعدل `03-data-model.md`.
- لو اتضاف widget جديد لازم يتذكر في `02-folder-map.md` و `04-runtime-flow.md`.
